Demerit Points Assignment

Description of Assignment
  • Problem: input a speed and calculated the demerit points
  • Note: speed limit is 70 km per hour
  • Note: for every 5km above speed limit, 1 demerit point
  • Note: at any speed limit, 12 demerit points means lisence suspended
  • Test cases:
    • 68 km/h: 0 pionts
    • 70 km/h: 0 points
    • 71 km/h: 0 points
    • 74 km/h: 0 points
    • 75 km/h: 1 Points
    • 85 km/h: 3 points
    • 102 km/h: 6 point (round down)
    • 115 km/h: 9 point
    • 130 km/h: lisence is suspended
  • Coding Hints
    • math.floor(): function to remove any decimal places, never rounding
